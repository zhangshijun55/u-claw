3 步搞定 3 Steps

双口 U 盘（USB-A + USB-C），Mac / Windows / Linux 通用 Dual-port USB (USB-A + USB-C), works on Mac, Windows & Linux

1

插入 U 盘

Plug in USB

双口 U 盘，USB-C 插 MacBook，USB-A 插 Windows 电脑

Dual-port USB: USB-C for MacBook, USB-A for Windows PC

2

双击启动菜单

Double-click Launcher

Mac 双击「启动菜单.command」，Win 双击「启动菜单.bat」，Linux 运行「运行.sh」

Mac: "启动菜单.command", Win: "启动菜单.bat", Linux: "运行.sh"

3

选择安装，搞定

Select Install, Done

选 [1] 一键安装，2 分钟后在终端输入 openclaw 即可使用

Select [1] One-click install. 2 minutes later, run "openclaw" in terminal

$ openclaw gateway restart
